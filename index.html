SVG Face and Scales

A scale is a function that maps an input domain (the range of possible input values) to an output range(the range of possible output values). Scales can be very useful for visualizations, if you had a scatterplot scales would allow you to reverse the output range so you could then see the same scatterplot with inverted x or y scales.Using scales also allows you to use d3.min() and d3.max() to very efficiently analyze your data.
